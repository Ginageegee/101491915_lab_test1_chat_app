## Overview
This is a real-time chat application built with:
- **Backend:** Node.js, Express, Socket.io, MongoDB (Mongoose)
- **Frontend:** HTML5, CSS, Bootstrap, jQuery, fetch

## Features
- User Signup
- User Login 
- Room-based chat (users can join/leave rooms)
- Real-time messaging using Socket.io
- Typing indicator
- Message persistence in MongoDB:
    - Group messages stored by room
    - Private messages stored between users
- Logout 

## Tech Stack
- Express
- Socket.io
- MongoDB + Mongoose
- Bootstrap 5 + jQuery
